## Overview

ZeroStarter includes optional integration for collecting user feedback. When configured, a feedback link appears in the docs sidebar, allowing users to submit feedback.

## Setup

### 1. Configure Environment Variable

Add your feedback URL to your `.env` file:

```
# Optional: Manage user feedback `https://zerostarter.dev/docs/manage/feedback`
NEXT_PUBLIC_USERJOT_URL=
```

### 2. Environment Configuration

The feedback URL is configured in `packages/env/src/web-next.ts`:

```ts
client: {
  NEXT_PUBLIC_USERJOT_URL: z.url().optional(),
}
```

The feedback link will automatically appear in the docs sidebar when `NEXT_PUBLIC_USERJOT_URL` is set. If the variable is not set or is empty, the link will not be displayed.

## Features

- **Sidebar Integration**: Feedback link appears in the docs sidebar under the "Support" group
- **Optional**: Only loads when `NEXT_PUBLIC_USERJOT_URL` is configured
- **External Link**: Opens feedback URL in a new tab

## Disabling Feedback

To disable the feedback link, simply remove or leave empty the `NEXT_PUBLIC_USERJOT_URL` environment variable. The link will not be rendered, ensuring zero performance impact when not in use.
